JSON tools allows you to diff objects, merge and overwrite one with another

Why would I need this?

If you like me, end up in a project where they use huge nested JSON objects for something. And you some times run into issues with version control, then this tool can help you make sure that you keep all of the properties.

Or if you want to get the diff between two objects in either direction.

Or if you want to just overwrite common values from one JSON object onto another.

The stack / Technology used

This app is created using Angular and is hosted in an AWS S3 bucket.
